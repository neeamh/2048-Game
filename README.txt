2048 Game

This repository contains a console-based implementation of the popular 2048 game in Python.

There are two files:
- 2048_pygame.py is the game with an actual interface
- 2048 low level which the IO version of the game(simplified)

Features:
- Console Interface: Play the game directly in your command line interface.
- Simple Controls: Use arrow keys to move tiles and merge numbers.
- Game Over Detection: Automatically detects when no more moves are possible.
How to Play:
- Clone the repository to your local machine.
- Navigate to the directory where the game is located.
- Run the game file (game.py, main.exe, etc.) to start playing.
- Use arrow keys (Up, Down, Left, Right) to move tiles.
- Combine tiles with the same number to reach 2048!
